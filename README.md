🥗 FOOD WASTE MANAGEMENT SYSTEM

📌 PROBLEM STATEMENT
A large amount of edible food gets wasted every day due to lack of proper distribution systems, while many people still struggle to get meals. This project solves this gap by connecting food donors with nearby NGOs and individuals who need food.

🚀 PROJECT OVERVIEW
The Food Waste Management System is a web-based platform that enables users to donate surplus food and allows receivers to request and collect it efficiently.
It uses location-based matching and real-time status tracking to reduce food wastage and improve food distribution.

⚙️ KEY FEATURES
👤 User Management
Separate login system for donors and receivers
Secure authentication (if implemented)
🍱 Food Donation System
Add surplus food details (type, quantity, expiry time, location)
Update and manage listings
🔁 Request & Approval Workflow
Receivers can request available food
Donors can approve or reject requests
📍 Location-Based Matching
Shows nearby food donations
Helps faster pickup and distribution
📊 Status Tracking
Available → Requested → Picked Up → Completed

🛠️ TECH STACK
Frontend: HTML, CSS, JavaScript
Backend: Node.js / Java / Spring Boot (update as per your project)
Database: MySQL / MongoDB
API: RESTful APIs
Tools: Git, GitHub, VS Code

🔄 SYSTEM WORKFLOW
Donor adds surplus food details
System stores food with location + expiry time
Receiver browses nearby listings
Receiver sends request
Donor approves request
Food is collected
Status is updated as completed
